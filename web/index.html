Available Roles

Super Developer

  • Able to use /nick (Change your name to anything)
  • Able to use /entereffect (Custom world entry effect)
  • Able to use secret commands (listed in /dhelp)
  • Cool orange name

Developer

  • Able to break bedrock in your own worlds
  • Ghost mode with /ghost command
  • One-hit block breaking with /1hit
  • Player teleport with /warpto
  • True invisibility with /invis
  • Must have Discord role to punish players

Moderator

  • Bypass world level limits
  • Player punishment abilities (requires Discord role)
  • Daily mod items (curse/fire wands & duct tapes)
  • Access to exclusive commands & early features

All In Pass

  • 100,000 gems, 5 exp potions instant bonus
  • 5x curse, fire & freeze wands
  • Permanent 2x deposit bonus
  • Stacks with other event bonuses

Cheater

  • Auto farm (1-3 slots) & Auto fish
  • Anti-bounce & Super-speed
  • Double jump & Heat resistance
  • Long punch & Auto gem pickup
  • Fast pull/drop/trash-recycle

Grow Pass

  • Monthly growpass rewards access
  • In-game /buy clothing command
  • Early access to new features

VIP

  • Rainbow name with /rainbow
  • Find players with /find
  • View inventories with /info
  • Quick teleport with /warp

Feature+

  • Access to /pay for gem transfers
  • Start business with /business
  • Beta server access with /beta
  • Vending search with /vending

SVIP

  • Access to exclusive commands
  • Unique feature access
  • Special privileges
